# DiaryUtility

Простая консольная программа на C# для работы с текстовым дневником.

Программа позволяет:

- создать или очистить файл дневника;
- добавить новую запись;
- прочитать все записи;
- узнать количество строк в дневнике.

Записи сохраняются в файле `diary.txt` в директории запуска программы.

## Требования

- .NET SDK 8.0 или новее.

Проверить установленную версию можно командой:

```bash
dotnet --version
```

## Запуск

1. Открыть терминал в папке проекта.
2. Выполнить команду:

```bash
dotnet run
```

3. Выбрать нужное действие, введя его номер в консоли.

Для завершения программы нужно выбрать пункт `0. Выход`.
